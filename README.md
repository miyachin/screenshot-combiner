# SnapBoard

Combine screenshots into beautiful phone mockups.

https://snapboard.lol

## Features

- Drag & drop up to 3 screenshots
- Phone bezel mockup with customizable thickness, color, size, and gap
- Multiple aspect ratios (16:9, 4:3, 1:1, 3:4, 9:16)
- Captions per screenshot
- Download as PNG (with or without transparent background)

## Development

```bash
npm install
npm run dev
```
